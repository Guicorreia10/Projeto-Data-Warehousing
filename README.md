# 📊 Data Warehouse & Business Intelligence – TechLusa

Projeto desenvolvido no âmbito do mestrado em Engenharia Informática, com foco na construção de um sistema de apoio à decisão baseado em Data Warehouse e Power BI.

---

## 🎯 Objetivo

Desenvolvimento de uma solução de Business Intelligence capaz de integrar, transformar e analisar dados de vendas, suportando decisões estratégicas.

---

## 🏗️ Arquitetura

Modelo dimensional em estrela composto por:

* Facto_Vendas
* Dim_Tempo
* Dim_Produto
* Dim_Cliente
* Dim_Localizacao

---

## 🛠️ Tecnologias

* SQL Server
* Data Warehouse
* ETL
* Power BI

---

## 📊 Análises desenvolvidas

* 📈 Análise temporal e sazonalidade de vendas
* 🛍️ Performance por categoria de produto
* 🌍 Distribuição geográfica (mapa de vendas)
* 👥 Ranking de clientes (Top N)

---

## 📌 Insights relevantes

* Identificação de picos de faturação (ex: janeiro e abril)
* Detecção de padrões sazonais
* Forte concentração de receita em poucos clientes
* Predominância de vendas em grandes centros urbanos

---

## 💡 Valor do projeto

Este projeto demonstra competências em:

* Modelação de dados
* Engenharia de requisitos
* Análise de dados
* Suporte à decisão

---

## 📂 Conteúdo

* Scripts SQL para criação e povoamento do Data Warehouse
* Modelo dimensional
* Relatório de análise
* Dashboard Power BI
